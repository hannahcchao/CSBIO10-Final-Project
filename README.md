# CSBIO10-Final-Project
Determining Most Impactful Characteristics for Breast Cancer Diagnosis

## FinalProjectProcessData.R contains the steps where the data was properly formatted and saved as a csv for downstream analysis

## FinalProjectAnalyzeData_PCA+EffectSizes.R contains the steps to create the PCA plot, violin plot comparing the distributions of the radii by diagnosis (with the p-value for the Wilcoxon test), and an effect size chart that compares the Cohen's d values of each dimension.
